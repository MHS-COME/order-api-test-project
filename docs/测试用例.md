## 3. 等价类划分

### 3.1 用户名（username）

| 等价类 | 范围 | 示例 |
|--------|------|------|
| 有效 | 4-32 字符 | `testuser` |
| 无效 — 太短 | < 4 字符 | `abc` |
| 无效 — 太长 | > 32 字符 | 33 位字符串 |
| 无效 — 空 | 空字符串或不传 | `""` |

### 3.2 密码（password）

| 等价类 | 范围 | 示例 |
|--------|------|------|
| 有效 | 6-64 字符 | `Test@123456` |
| 无效 — 太短 | < 6 字符 | `12345` |
| 无效 — 太长 | > 64 字符 | 65 位字符串 |
| 无效 — 空 | 空字符串或不传 | `""` |

### 3.3 购买数量（quantity）

| 等价类 | 范围 | 示例 |
|--------|------|------|
| 有效下边界 | `1` | `1` |
| 有效上边界 | `999` | `999` |
| 有效中间值 | 2-998 | `5` |
| 无效下边界 | `< 1` | `0` |
| 无效上边界 | `> 999` | `1000` |
| 无效 — 负数 | `< 0` | `-1` |

### 3.4 订单 ID（id）

| 等价类 | 范围 | 示例 |
|--------|------|------|
| 有效 | 存在且属于当前用户的正整数 | 创建后获取的 ID |
| 无效 — 不存在 | 有效的正整数但数据库中无记录 | `99999999` |
| 无效 — 非数字 | 非整数字符串 | `abc`, `1.5` |
| 无效 — 非正数 | 0 或负数 | `0`, `-1` |
